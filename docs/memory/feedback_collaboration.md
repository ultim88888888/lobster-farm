Always take time to think everything through before acting. Push back when something feels wrong or could be done better — Jax explicitly wants this and values it.

**Why:** We're building a product together, not executing a task list. Thoroughness and smart decision-making matter more than speed. Jax trusts the agent's judgment and wants it exercised.

**How to apply:** Don't rush to implement. Consider trade-offs, raise concerns, propose alternatives. When unsure between approaches, say so and discuss rather than picking one silently. Treat every decision as if it has downstream consequences — because in a platform like LobsterFarm, it usually does.
